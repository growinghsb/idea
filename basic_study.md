# 기초 공부 시 만들어 볼 간단한 프로그램들     
1. 학생 성적 입력 프로그램(학점 입력 프로그램)      
2. 구구단    
3. 계산기      
4. 야구게임      
5. 전화번호부 프로그램      
6. 입력 받은 알파벳별 개수 세기      
7. 계단, 삼각형, 다이아몬드, 원 찍기      
8. 잠금장치 프로그램(비밀번호를 이용한)       
9. 가위바위보 게임        
10. 경마 게임(레이싱 게임 변형)     
11. 끝말잇기 게임           
12. 최근 검색 목록 만들기(검색어 입력하면 최근 검색 순대로 보여주기)
